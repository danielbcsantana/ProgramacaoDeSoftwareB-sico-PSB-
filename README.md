# Programacao de Software Basico (PSB)

Circuitos e códigos da matéria - Programação de Software Básico (UFBA - 2025.2)

# Informações Relevantes:

- Nesse repositório foram desenvolvidos códigos e circuitos.

  Os códigos estão desenvolvidos a partir do Assembly AVR (conjuntos de instruções Assembly para Microcontroladores da família AVR)
  
  Os circuitos estão desenvolvidos para o Microcontrolador ATmega328p (o microcontrolador presente no Arduino UNO)
- Os circuitos foram desenvolvidos utilizando o programa SimulIDE e tem extensão ".sim1"
- Os códigos foram desenvolvidos utilizando o programa Microchip Studio e tem 2 tipos de extensão:
  
  ".asm" (código sem compilar), ".hex" (código compilado pronto para ser utilizado no SimulIDE)
- Para mais informações acesse esse documento: asdasdasjkldjlasd
